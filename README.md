# codeprint
- print code in jupyter notebook


## Usage
- Import
```python
from codeprint import RawCode
```
- define RawCode Class
```python
code = RawCode('codeprint.py')  # put code name
code.class_info 3                # returns all classes in code
```

- print code
```python
code.show('RawCode')
```
